
## ¿Qué es un DAG?

Un DAG, que significa **Directed Acyclic Graph** (Grafico Acíclico Dirigido), representa una colección de tareas, donde cada tarea está conectada y estructurada de manera que no haya ciclos. Esto garantiza que cada tarea solo se ejecute una vez.

## ¿Qué es un Parquet?

Parquet es un formato de almacenamiento de datos en columnas que es ampliamente utilizado en el ecosistema de big data. En el contexto de Python, se utiliza sobre todo en conjunción con bibliotecas de manejo de datos como Pandas y PyArrow.

## ¿Qué es un DuckDB?

DuckDB es un sistema de gestión de bases de datos SQL que se especializa en la consulta de datos, especialmente en entornos de análisis de datos. Está diseñado para ser ligero y fácil de usar, ofreciendo un enfoque de almacenamiento en memoria que permite un rendimiento rápido en consultas complejas.
